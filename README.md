Sobre o Projeto

O Micro$ales é um sistema web voltado para organização e acompanhamento de vendas, pedidos e despesas, funcionando como um mini CRM com foco em controle financeiro e gestão de pipeline. A aplicação permite que o usuário registre leads, acompanhe o status de pedidos em diferentes etapas, visualize indicadores de desempenho no dashboard e monitore receitas e gastos mensais. Os dados são armazenados em banco de dados por meio do Turso DB, enquanto o acesso ao sistema é controlado com autenticação segura via Clerk.

A homepage apresenta um painel de visão geral com métricas essenciais como vendas do dia, pedidos pendentes, lucro do mês e gastos do mês, além de gráficos que demonstram evolução de receita e distribuição de pedidos por etapa. A organização das informações foi pensada para oferecer leitura rápida e compreensão imediata, mesmo para usuários sem conhecimento técnico.

A interface foi construída com foco em clareza, hierarquia visual e linguagem acessível. Termos técnicos foram simplificados para facilitar o entendimento por usuários leigos, mantendo ao mesmo tempo a precisão dos dados apresentados. O design prioriza organização, contraste adequado entre temas claro e escuro e uso consistente de componentes, garantindo uma experiência profissional, funcional e intuitiva.
